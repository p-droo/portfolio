# Meu Portfólio Pessoal 

Este é o repositório do meu portfólio pessoal hospedado em [https://p-droo.github.io/portfolio/ ](https://p-droo.github.io/portfolio/ ).

Desenvolvido com React e TailwindCSS, esse site é minha vitrine digital onde compartilho projetos, habilidades e formas de contato.

## 💡 Sobre mim

Profissional atuando na interseção entre educação, inovação e desenvolvimento de produtos digitais. Minha trajetória reúne experiência em ensino, curadoria de conteúdo, design de projetos educacionais e empreendedorismo — focado agora na criação de soluções de aprendizagem acessíveis e estratégicas.

## 🛠️ Tecnologias usadas
